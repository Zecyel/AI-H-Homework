## 2. 第一部分:反向传播算法 (60%)

### 2.1 任务内容

本部分要求基于反向传播算法，自行设计并实现神经网络模型，完成以下两类任务：

| 任务类型 | 具体要求 |
| :--- | :--- |
| 回归任务 | 拟合函数$y=sin(x)$，其中$x\in[-\pi,\pi]$。 |
| 分类任务 | 对12个手写汉字进行分类。 |

### 2.2 具体要求与评分构成

| 评分项 | 要求说明 |
| :--- | :--- |
| (50分)网络结构设计与实现 | 需设计并实现一个具有良好可扩展性与可调节性的网络结构，例如能够灵活设置层数、每层神经元数量、学习率等超参数。 |
| (20分)任务性能要求 | 需在回归任务与分类任务上达到规定精度要求。其中，正弦函数数据样例需自行生成，并要求在指定区间内进行随机采样，平均误差小于0.01。手写汉字分类任务的训练数据集需从 elearning下载，分类精度分数将依据面试环节的精度排名确定。 |
| (30分)实验文档 | 需提交完整实验文档，内容可包括但不限于：代码基本架构说明；不同网络结构与参数设置的实验比较；对反向传播算法原理的理解与分析等。文档应工整、详实、美观，并导出为PDF格式。 |

### 2.3 实验建议

* 回归任务与分类任务所采用的损失函数并不相同，建议在实现时注意区分，并思考这种差异背后的原因。
* 在训练初期，可适当将权重初始化值与学习率设置得较小，并将 bias 初始化为负数，以帮助模型更稳定地收敛。
